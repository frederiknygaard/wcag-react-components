# Focus

The focus component is a wrapper for react components, that enables features like using the arrow buttons, home-button and end-button

## Usage

```javascript
import { Focus } from 'wcag-react-components'
```

```javascript
  <Focus focusableElements={[document.getElementById("myButton"), ...]} >
    <div>
      <button id="myButton">My Button</button>
    </div>
  </Focus>
```
