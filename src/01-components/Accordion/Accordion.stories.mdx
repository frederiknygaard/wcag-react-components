# Accordion

The accordion is based upon https://www.w3.org/TR/wai-aria-practices-1.1/examples/accordion/accordion.html

## Usage

```javascript
import { Accordion } from 'wcag-react-components'
```

```javascript
<Accordion>
  <Accordion.Item label="Accordion Trigger #1">Accordion Content #1</Accordion.Item>
  <Accordion.Item label="Accordion Trigger #2">Accordion Content #2</Accordion.Item>
  <Accordion.Item label="Accordion Trigger #3">Accordion Content #3</Accordion.Item>
</Accordion>
```
