# Breadcrumb

The breadcrumb is based upon https://www.w3.org/TR/wai-aria-practices/examples/breadcrumb/index.html

## Usage

```javascript
import { Breadcrumb } from 'wcag-react-components'
```

```javascript
<Breadcrumb
  aria-label="Breadcrumb"
  items={[
    {
      children: 'Breadcrumb item #1',
      href: '#',
    },
    {
      children: 'Breadcrumb item #2',
      href: '#',
    },
    {
      children: 'Breadcrumb item #3',
      href: '#',
    },
  ]}
/>
```
